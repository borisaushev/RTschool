## SQUARE EQUATION SOLVER
### By BossBoriss

-------------------
Build: 
```bash
  make release
```
Run:
```bash
  .\release.exe
```

If you want to run on data from file, you need a file with such structure:
```text
    a1 b1 c1
    a2 b2 c2
    ...
```
Run with data from file
```bash
  .\build.exe --FILE <your_file_path>
```

Build and run with tests:
```bash
  make test
```
```bash
  .\test.exe
```

Build and run with flags:
```bash
  make build
```
```bash
  .\build.exe
```
